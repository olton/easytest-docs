The console reporter is the default reporter used by Latte. 
It displays test results in the console, providing a quick and easy way to see the status of your tests.

## Usage

To use the console reporter, simply run your tests with the `--coverage --reportType=console` options set. For example:

```bash
latte --coverage --reportType=console
```
