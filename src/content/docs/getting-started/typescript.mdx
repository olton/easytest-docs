Latte can be used with TypeScript and for TypeScript projects.

## ![Babel](../../../assets/TypeScript.svg) TypeScript Support

To use Latte with `TypeScript` you need to install `babel`:

<Tabs>
    <TabItem label="npm" icon="seti:npm">
        ```bash
        npm install -D @babel/core @babel/preset-env @babel/preset-typescript
        ```
    </TabItem>
    <TabItem label="pnpm" icon="pnpm">
        ```bash
        pnpm -D install @babel/core @babel/preset-env @babel/preset-typescript
        ```
    </TabItem>
    <TabItem label="yarn" icon="seti:yarn">
        ```bash
        yarn add -D @babel/core @babel/preset-env @babel/preset-typescript
        ```
    </TabItem>
    <TabItem label="bun" icon="bun">
        ```bash
        bun add -D @babel/core @babel/preset-env @babel/preset-typescript
        ```
    </TabItem>
</Tabs>

## Create Babel Configuration
Create a new file named `.babelrc` in your project directory and add the following code:

```json title=".babelrc"
{
    "presets": [
        "@babel/preset-env",
        "@babel/preset-typescript"
    ]
}
```

## Create your first test
Create a new file named `example.test.ts` in your project directory and add the following code:

```ts title="example.test.ts"
function hello(): string {
    return "Hello"
}

describe(`Common tests suite`, () => {
    it(`says hello`, () => {
        return expect(hello()).toBe("Hello")
    })
})
```
